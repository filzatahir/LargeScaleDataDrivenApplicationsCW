# Large Scale Data Driven Applications – Coursework Project

A compact, well‑structured repository showcasing data preprocessing, modelling, and dashboard creation for the **Large Scale Data Driven Applications** module.

---

## 📁 Project Structure
```
├── Dashboard_Project/        # Dashboard files & assets
├── Data/
│   └── ERD.png               # Entity Relationship Diagram
├── Preprocessing.ipynb       # Data cleaning & preprocessing workflow
├── README.md                 # Documentation
└── ~$r Dataset Analysis.docx # (Autosave file)
```

---

## 🚀 Features

### **🔧 Preprocessing Notebook**
Located in `Preprocessing.ipynb`, this notebook covers:
- Data cleaning  
- Handling missing values  
- Feature engineering  
- Exploratory data analysis  
- Preparation for dashboarding and modelling  

### **📊 Dashboard**
The `Dashboard_Project/` folder contains the dashboard built for the coursework, visualising key insights and trends derived from the dataset.

### **🗃️ ERD**
`Data/ERD.png` provides a clear overview of the dataset structure and relationships, supporting database design and analysis.

## 🛠️ Getting Started

### **Requirements**
- Python 3.8+
- Jupyter Notebook / JupyterLab  
- Common data libraries:
  - pandas  
  - numpy  
  - matplotlib / seaborn  
  - (Plus any dashboard‑specific dependencies)

### **Run the Notebook**
```bash
git clone https://github.com/filzatahir/LargeScaleDataDrivenApplicationsCW
cd LargeScaleDataDrivenApplicationsCW
jupyter notebook
```

Open `Preprocessing.ipynb` and run all cells.
